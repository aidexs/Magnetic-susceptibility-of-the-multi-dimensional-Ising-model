# 图表使用说明

## 生成的图表文件

1. **m_vs_t_comparison.pdf** - 对应论文中的图1
   - 展示1-4维伊辛模型磁化强度随温度变化对比
   - 不同颜色和标记代表不同维度
   - 清楚显示临界温度随维度增加的趋势

2. **physics_analysis.pdf** - 补充分析图
   - 包含理论临界温度的垂直线标记
   - 添加了物理解释文本
   - 可用于论文附录或补充材料

3. **beta_theoretical.pdf** - 对应论文中的图3
   - 展示临界指数β的理论值随维度变化
   - 柱状图形式，便于比较

4. **critical_parameters.pdf** - 对应论文中的图2
   - 同时展示临界温度和临界指数随维度变化
   - 左右两个子图分别显示Tc和β

## 数据文件引用

论文中提到的具体数值可以从以下数据文件中获得：

- **m_vs_t_1d.dat, m_vs_t_2d.dat, m_vs_t_3d.dat, m_vs_t_4d.dat** - 磁化强度数据
- **m1_beta_star_1d.dat, m1_beta_star_2d.dat, m1_beta_star_3d.dat, m1_beta_star_4d.dat** - β*分析数据

## 论文中的图表引用

在正文中，图表应该这样引用：

```
如图1所示，1-4维伊辛模型的临界温度随维度增加而单调递增...

表1总结了不同维度下的理论预测值与我们的模拟结果...

从图3可以看出，临界指数β随维度增加逐渐趋近于平均场论值0.5...
```

## 数据处理说明

所有图表都是基于实际的蒙特卡洛模拟数据生成，确保了结果的可靠性。模拟参数在论文的表1中有详细说明，包括晶格大小、热化步数、测量步数等。
